# Pygovue

Pygovue is a 3-part website skeleton, Python/ Django acting as a backend, Golang as an API and Vuejs 2 as a frontend.

## Status

In dev.

## Requirements

Python 3.6, Golang 1.8, Vue 2/ Nuxt, Postgres.

## Setup

1. Create database.
2. Edit .env
3. Run the build:

```
./build.sh <PROJECT_NAME>
```
